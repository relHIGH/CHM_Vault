# 陶瓷
仰韶彩陶盆
（新石器，西安半坡）：人面鱼纹。 _原始宗教、彩陶工艺_

###### 汉画像石
（东汉为主，山东/河南/陕北/苏北）：用于汉代地下墓室、墓地祠堂及墓阙等建筑上的石刻艺术，被誉为“绣像的历史”。
*形制工艺：主要采用平雕、线刻、凹雕（阴线刻）等手法。
核心题材：社会生活（车马出行、乐舞百戏）、历史故事（二桃杀三士、周公辅成王）、神话传说（西王母、东王公、升仙图）。
学术价值：实证了汉代“事死如事生”的丧葬观念；是研究汉代建筑、服饰、礼俗及美术史的百科全书；反映了汉代由神仙思想向世俗生活转变的审美逻辑。* 
###### 越窑
（东汉—宋，浙江绍兴/上虞）：中国古代最著名的青瓷窑系，开创了“南青北白”中南方的制瓷巅峰。
*核心特征：釉色类玉类冰（秘色瓷为巅峰）、装饰以刻划花为主、采用龙窑烧制及匣钵装烧工艺。
代表器物：鸡首壶、莲花尊、唐代“秘色瓷”。
学术地位：中国成熟青瓷的源头，六朝时期通过漕运与海外贸易影响深远，史称“瓷都”。* #陶瓷科技考古  #六朝考古/六朝青瓷

###### 苏州宋锦
（宋—明，江苏苏州）：活化石。 _织锦工艺改良、江南手工业地位_ #长江下游
    
###### 江宁织造遗存
（清代，江苏南京）：皇家贡品。 _云锦工艺、清代内务府管理制度_ #长江下游 #南京地域 
###### 良渚丝织品
（新石器晚期，浙江钱山漾/反山）：实证中国家蚕丝织造起源的核心物证。
*核心发现：湖州钱山漾遗址出土的绢片、丝带和丝线（中国最早的家蚕丝织品）；余杭反山墓地（如14号墓）出土的玉器上附着的丝织品残迹。
工艺特征：采用家蚕丝，已掌握平纹织造技术（经纬交织），部分丝线经过加捻处理。
学术价值：证明了良渚先民已完成从野蚕到家蚕的驯化；丝绸作为高级奢侈品，在当时与玉器共同构成了象征身份等级的“礼制系统”。* #长江下游
	
######  吴越宋元丝织品
（两宋—元代，江浙地域）：中国丝绸工艺巅峰，确立江南“丝绸之府”地位。
*核心品种：缂丝（织中之圣）、罗（轻薄透孔）、加金锦（纳石失）。
工艺特征：宋代盛行“通经断纬”缂丝；元代流行“织金”技术（大规模使用金线）。
代表发现：苏州虎丘塔（宋缂丝）、福州黄昇墓（宋服饰大全）、南京幕府山元墓。
学术价值：实证经济重心南移，反映蒙元时期东西方工艺交流。* #长江下游
