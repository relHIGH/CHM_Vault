# 三星堆遗址
[[遗址#三星堆遗址]]
- 新阶段发掘（2021–2023）
    → 十大 + 新成果，方法论与文明解释双重价值
- **“多元一体”的证据**：
    
    - **异**：面具、神树、金杖（独特的蜀地宗教审美）。
        
    - **同**：青铜尊、罍（受商文化影响），玉璋（受二里头文化影响）。
        
- **祭祀坑的性质**： 目前学术界主流观点倾向于这些坑是“毁宝祭祀”或“改朝换代的埋藏”，而非简单的随葬墓穴。
    
- **技术考点**： 三星堆青铜器采用了**分铸法**（如神树的枝干连接），且其合金成分中铅含量较高，体现了区域性的工艺特征。

# 石峁遗址
[[遗址#石峁遗址]] [[文物692+898#2024]]
- 城防—祭祀—王权结构再认识
	- **城防鼻祖**： 石峁东门址发现了**瓮城、马面、角楼**。这些原本被认为出现在汉唐之后的防御设施，在4000多年前的石峁就已经成熟。
	    
	- **“藏玉于墙”**： 石峁先民在修筑城墙时，将玉铲、玉璜等塞入墙缝中。这被认为是一种**建筑祭祀**行为，寓意“玉石镇城”。
	    
	- **皇城台阶层**： 皇城台高达数十米，这种**“台墩式”核心区**不仅是权力的象征，也体现了极强的社会动员能力。
- 古DNA揭示新石器遗传关系
# 凌家滩遗址
- 玉礼制与早期文明等级
    [[遗址#凌家滩遗址]]
    - **玉版八角星纹**： 这是凌家滩的“灵魂”。玉版上的刻划符号被认为是**原始历法或方位系统**的雏形，反映了当时先民对天文宇宙的认知。
    
- **与良渚的关系**： 凌家滩早于良渚。良渚中常见的**三层坛式祭坛**、**玉琮雏形**以及某些纹饰，在凌家滩都能找到源头。考古界常有“凌家滩外传”影响良渚的说法。
    
- **社会分化**： 墓葬中“一人独尊”的随葬现象（有的墓随葬数百件玉器），说明当时已进入了**“贵玉时代”**，王权与神权高度结合。

# 西夏陵
[[不可移动#西夏陵]]
- 多学科综合与遗产阐释,城址布局与民族融合。
- **“汉制”与“党项制”的结合**
    - **汉制**：轴线对称、碑亭、献殿、石像生。
    - **党项/佛教特征**：墓冢（灵台）不在建筑群中心而是偏北，且形状像**佛塔**（夯土层层内收）。
        
- **西夏文碑刻**： 陵区出土了大量**西夏文与汉文合璧**的碑残片，是破译西夏文字、研究西夏官制的核心资料。
    
- **迦陵频伽（妙音鸟）**： 这是西夏陵最标志性的建筑装饰，这种人首鸟身的佛教形象被大量用于建筑构件，反映了西夏“以佛治国”的统治特点。

## 青墩遗址（或草鞋山等相关新发现）
（新石器时代早期，江苏）：苏南地区发现距今约8100年的文化遗存，被誉为“最早的江南”。
*核心发现：出土早期陶器及聚落遗迹，将江南地区的文明起步时间向前大幅推移。
学术价值：实证了长江下游地区史前文化的连续性，是研究江南地区生业模式与聚落演变的新起点。* #南京地域

## 稽山中学越国祭祀遗址
（春秋战国，浙江绍兴）：绍兴城内首次确认的越国大型高等级祭祀建筑基址。
*核心发现：发现巨型木结构建筑遗迹及与祭祀相关的器物组合，位于古越国核心区。
学术价值：填补了越国高等级祭祀遗存的考古空白，对研究越国都城布局及宗教礼仪具有极高价值。*

## 丁公遗址（或相关遗址新进展）
（新石器时代，山东）：在早期地层中发现距今约9000年的炭化小豆。
*核心发现：通过植物考古浮选法获得极早期的豆类炭化标本。
学术价值：将中国豆类作物的利用历史推至万年门槛，为研究海岱地区农业起源及早期食谱提供了珍贵实物。*
