# 考古学文化的定名

[[-答题模版-#简答题]]

> [!example] 简答
考古学文化的定名是文化历史考古学分类体系的基础，夏鼐在《关于考古学上文化的定名问题》（1959）中系统提出命名三原则，解决了物质文化分类标准化的核心问题。
> 
> - **地域命名优先**：以首次发现的典型遗址的小地名（村镇名）作为文化名称，如"仰韶文化"（河南渑池仰韶村）、"龙山文化"（山东济南龙山镇）。避免使用大地域名（如"河南文化"）或现代行政区划。
>     
> - **时间维度限定**：当同一地区存在不同时期文化时，需加时间限定词区分，如"老官台文化"早于"仰韶文化"，或区分为"早期龙山文化""晚期龙山文化"。防止将与历史时期民族文化简单对应（如避免称"夏文化"而称"二里头文化"）。
>     
> - **典型遗址原则**：选择遗存丰富、特征鲜明、发掘规范的遗址作为命名依据，而非偶然发现或单一看似典型器物。要求具备一定数量的遗迹遗物组合，体现文化的整体特征而非孤立因素。
> 
> 以"良渚文化"为例，1959年夏鼐以浙江余杭良渚镇遗址群命名，取代了此前"龙山文化杭州湾类型"的笼统称谓，体现了由类型到独立文化、由泛称到专称的学术进步。
> 
> 该定名体系奠定了中国考古学文化编年的基础，但需注意文化命名后的"标签化"倾向，避免将考古学文化与古代族群简单等同。
> 

- 依据
	- 首次发现典型遗址的小地名
		- 要求
			有一批能够代表文化的遗迹遗物
			在中心地区
			延续时间能够cover主要阶段
			有一定规模的发掘
	- 代，国家，古族命名
	- 与同时代临近的族的共同体的关系
	- 典型的文化特征

> - 已停用定名方法
> 	使用特征性遗物,如黑陶文化
> 	加前缀 如 浙江 龙山文化,改用小地名——良渚文化
> 	加后缀 如xx二期，下层

- 要注意
	- 文化的分布范围
	- 文化面貌及其特征
	- 文化内涵所反应的社会形态
	- 文化过渡地区


# 区系类型理论

[[-答题模版-#名词解释]]

> [!tip] 名词解释
>  #区系类型理论 是由中国考古学家苏秉琦于20世纪70年代%%1975年提出、1981年完善%%提出并完善的中国新石器时代考古学文化谱系研究框架。其核心内涵是以考古地层学与类型学为基础，通过“区”（空间布）、“系”（时间演进）、“类型”（文化分支）三个维度，系统构建文化的时空框架。
> 
> 1. **提出背景**：由苏秉琦在《关于考古学文化的区系类型问题》（1981）一文中系统阐述，旨在突破“中原中心论”。
> 2. **内容方法**：以文化谱系分析为核心方法，强调对文化进行横向分区与纵向分系，划分出六大文化区系（如北方、中原、东方等）及下属类型, 认为中华文明并非单一起源、向外辐射，而是六大区系独立发生发展、相互影响融合的多元一体格局。
> 3. **研究对象**：主要适用于中国新石器时代至青铜时代早期的考古学文化研究，旨在揭示文化之间的源流、交流与谱系关系。
> 
> 该理论标志着中国考古学从单一文化描述转向系统谱系研究，为理解中华文明“多元一体”格局提供了必要的时空框架基础和关键分析工具，是中国考古学从材料积累转向系统复原古史的重要标志, 具有奠基性意义。但其侧重于文化表象的归类与关联，对社会内在动力与过程的解释尚显不足。


>考古遗存单位➡️聚落组成单位➡️遗址➡️区域
>共时关系 - 先后关系 - 功能关系 

* 新石器时代至青铜时代早期 **六大区系**：
	1. 以燕山南北长城地带为中心的北方（北方区系）
	    
	2. 以山东为中心的东方（东方区系）
	    
	3. 以关中（陕西）、晋南、豫西为中心的中原（中原区系）
	    
	4. 以环太湖为中心的东南部（东南区系）
	    
	5. 以环洞庭湖与四川盆地为中心的西南部（西南区系）
	    
	6. 以鄱阳湖—珠江三角洲为中轴的南方（南方区系）

# 文化因素分析

[[-答题模版-#名词解释]]

> [!tip] 名词解释
> 
> **【学术定义句】**  
>  #文化因素分析 是20世纪80年代兴起于中国考古学的研究方法，指通过解剖考古学文化内部不同文化因素的来源、构成比例及互动关系，以揭示文化形成机制与人群交流模式的系统性分析框架。
> 
> **【核心要素拆解】**  
> 
> - **时间/人物**：1980年代由严文明在《考古学报》相关论述中系统阐述，借鉴张光直"相互作用圈"理论并结合中国考古实践发展而来；
>     
> - **内容/方法**：以文化因素分类（传承因素、外来因素、混合型因素）为核心，运用量化统计（百分比统计、分布态势图）与类型学比较，强调文化因素的层级划分与源流追溯；
>     
> - **对象/范围**：主要适用于文化谱系复杂区域（如黄河流域、长江中游），涵盖文化起源、迁徙、融合及边缘区域文化互动等问题的研究。
>     
> **【学术定位句】**  
> 该概念标志着中国考古学从文化史描述向社会考古与人群关系研究的深化转向，对解释两周考古学文化谱系、文明形成机制具有方法论意义，但存在因素定性主观性强、统计标准不一等操作局限。
>
