# 漆
胎
	木，布，竹，金属，陶
	夹纻胎
		木或泥作内胎，涂漆灰的麻布糊若干层，后去掉内胎
乾隆发展到顶峰
工艺
	制胎—髹漆—装饰（描金、螺钿）（层次与工序）
病害核心：  
	漆层起翘、开裂、剥落，胎体变形
保护要点：  
	控制湿度、防漆膜应力变化，修复以**加固为主**
	宗旨是保持现状而不是优化美化
漆器修复
	清洗-除菌-除霉-脱水加固-草酸除污脱色

# 木雕竹刻
竹
	金陵派
		浅刻法
	嘉定派
		深刻法
	留青
	贴黄
- 材料特性：  
    中空纤维结构，弹性强但易纵裂
    
- 工艺重点：  
    剖篾、编结、弯曲定型
    
- 病害核心：  
    纵向开裂、脆化、虫蛀
    
- 保护要点：  
    防干裂、防生物病害，避免拉伸
木
- 材料特性：  
    纤维定向明显，**吸湿胀缩强**
    
- 工艺重点：  
    削制、榫卯、拼接
    
- 病害核心：  
    干缩、翘曲、开裂、虫蛀
    
- 保护要点：  
    缓慢控湿、防虫，结构支撑优先


# 骨器象牙器犀角器

- 材料特性：  
    动物硬组织，成分接近但**结构与稳定性不同**
    
- 工艺重点：  
    切割、打磨、抛光（非冶炼）
    
- 病害核心：  
    脱水、粉化、层裂
    
- 保护要点：  
    稳定环境，避免清洗与脱脂过度
# 其他物件
## 珐琅
- 材料属性：  
    金属胎＋玻璃质釉层的**复合材料**
    
- 工艺核心：  
    施釉→烧成（掐丝/画珐琅等）
    
- 研究重点：  
    工艺来源、技术成熟度、中西交流
    
- 病害关键：  
    釉裂、剥釉、胎釉分离
    
- 保护要点：  
    控温差、防震，避免釉层应力变化
## 核雕
明清兴盛
- **考点**
    
    - 极限减材工艺（微雕水平）
        
    - 文人趣味与手工艺等级
        
    - 有机质材料稳定性问题
        
- **病害与保护**
    
    - 病害：干裂、脆化、油污渗入
        
    - 保护：恒温恒湿、防手触、防油脂，禁止清洗与上油
## 鼻烟壶
康熙出现
- **考点**
    
    - 清代生活方式与社会风尚
        
    - 多材料综合工艺（器类而非单一材质）
        
    - 中西交流背景下的工艺融合
        
- **病害与保护**
    
    - 病害：随材质不同（玻璃风化、珐琅釉裂、玉石沁蚀等）
        
    - 保护：按材质分类管理，防震、防温差，避免混用修复方法

> [!example] 简答
>工艺类文物对制度研究的意义在于：**通过生产组织、使用等级与资源分配，反映制度在日常层面的实际运行方式。**
> 
> **T｜定位**  
> 工艺类文物并非制度器物本身，但其制作与使用高度依赖社会组织与规范，因此是研究制度如何落实到技术与生活层面的重要实物材料。
> 
> **R｜分述**  
> 第一，工艺类文物反映**制度对生产的组织能力**。如漆器、织绣等制作周期长、工序复杂，往往需要稳定的工匠体系与资源调配，体现国家或权力集团对人力与物资的制度化管理。  
> 第二，工艺类文物体现**制度对使用等级的规范作用**。材料选择、工艺复杂度与装饰程度常与身份等级相对应，反映礼制与等级制度在物质层面的具体约束。  
> 第三，工艺类文物揭示**制度渗透日常生活的方式**。相较于法律文书或礼器，工艺制品更贴近日常使用，其差异化呈现制度在实际生活中的执行深度与弹性。
> 
> **E｜例证**  
> 如高等级漆器、多层髹漆与复杂装饰的集中出现，说明其生产与使用受到制度性限制；织绣、骨角牙器物中材料与工艺的等级差异，也反映制度对资源与技艺的分配逻辑。
> 
> **E｜小结**  
> 因此，工艺类文物为制度研究提供了从“技术与日常实践”切入的视角，是理解制度如何被组织、执行并内化于社会生活的重要补充材料。
>

# 保护与修复
**常见病害（高频）**

- 干缩、翘曲、开裂
    
- 脱水、粉化、虫蛀
    
- 漆层起翘、剥落
**保护与修复原则**

- 稳定优先，修复后置
    
- 恒温恒湿、避光、防虫
    
- 最小干预、可逆性

常用技术
**1️⃣ 显微与结构观察（最常用）**

- 体视显微镜、偏光显微镜
    
- 看什么：
    
    - 纤维走向（木/竹）
        
    - 胶原结构（骨角牙）
        
    - 漆层叠压关系（漆器）
        
    - 刀痕与工艺顺序（核雕）  
        👉 用于**工艺判断、病害机理分析**
        

---

**2️⃣ 无损成像（比成分更重要）**

- X 光、CT
    
- 看什么：
    
    - 内部裂隙、虫道
        
    - 胎体与表层关系
        
    - 鼻烟壶内绘、内腔结构  
        👉 解决“看不见但会出问题”的部分
        

---

**3️⃣ 劣化机理分析（近年重点）**

- 不是“是什么做的”，而是
    
    - 为什么会裂
        
    - 为什么会粉
        
    - 为什么会起翘
        
- 针对：
    
    - 木材吸湿循环
        
    - 胶原水解
        
    - 漆膜应力失衡  
        👉 用于**指导保护方案**
        

---

**4️⃣ 成分分析（有限使用）**

- FTIR、Raman、XRF
    
- 主要用途：
    
    - 区分漆 / 胶 / 釉 / 金属
        
    - 判断修复材料、后加物  
        ⚠️ **不是常规手段，更不是第一手段**
        

---

**5️⃣ 数字化技术（新趋势）**

- 3D 扫描、数字建模
    
- 用途：
    
    - 记录微雕细节
        
    - 修复前方案模拟
        
    - 减少实际操作次数
