# 石窟
（作为空间与载体）  
石窟是壁画和彩塑的**依附空间**。  
研究重点：

- 洞窟形制与布局
    
- 功能分区（礼拜、供养）
    
- 与宗教传播、政权支持的关系
    

👉 石窟本身＝**宗教制度的空间化**。
类型
	塔庙窟  佛殿窟  -有无中心塔柱
	大像窟   佛坛窟 -里边供的佛像
	僧房窟    小型禅窟/罗汉窟（群）

# 壁画
（图像系统）  
壁画是宗教叙事与观念传播的主要媒介。  
研究重点：

- 题材类型（佛传、本生、供养人）
    
- 图像组合逻辑
    
- 与时代审美、宗教需求的关系

# 彩塑
（立体形象）  
彩塑是最具现场感的宗教形象。  
研究重点：

- 塑像组合与等级
    
- 空间中的位置关系
    
- 崇拜与仪式功能
    

👉 彩塑体现的是：  
**信仰对象的等级秩序与崇拜核心**。

# 研究意义
政权，宗教制度化，资源倾斜，阶层，艺术发展水平
供养人

# 断代与研究方法（共通）

- 洞窟形制
    
- 图像题材与组合
    
- 塑像风格
    
- 材料与工艺
    
- 文献与题记
    

科技只作：  
材料分析、病害判断、保护记录。


# 保护与修复

- **主要病害类型（点名即可）**
	
	- 自然风化：岩体剥落、裂隙发展
	    
	- 环境因素：温湿度波动、渗水、盐害
	    
	- 生物病害：微生物、植物
	    
	- 人为影响：游客活动、早期不当修复

- 保护策略
	- 强调原真性与整体性
	    
	- 防止过度修复与重绘
	- **预防性保护优于修复性保护**
	
	- 具体体现为：
	
		- 环境监测与控制（温湿度、水害）
		    
		- 限制人为干扰（参观管理）
		    
		- 数字化记录与长期监测
