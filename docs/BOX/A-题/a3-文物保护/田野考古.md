# 考古现场保护



> [!tip] 名词解释
**考古现场保护**是20世纪70年代后兴起的[[藏品保护#预防性保护|预防性保护]]分支，指在考古发掘过程中对出土脆弱遗存实施的**原位临时 stabilization** 与**应急处理**。它以**最小干预**、**可逆性**、**及时性**为原则，通过保湿、临时固型、整体提取等技术，防止文物因环境突变（温湿度、氧化、机械震动）发生**二次损伤**（如漆木器干裂、青铜器快速氧化），是博物馆藏品保护的前置节点。



> [!example] 考古现场保护简述
> 
> 考古现场保护是[[藏品保护#预防性保护|预防性保护]]（Preventive Conservation）在田野发掘中的首要实施环节，指在遗存脱离埋藏环境的**黄金时间窗**（出土瞬间至转移前），通过微环境控制与临时干预，防止文物因温湿度骤变、氧化或机械损伤发生**二次劣化**的技术体系。
> 
> **核心原则**
> - **最小干预**：仅维持稳定，不做深度修复
> - **可逆性**：临时固型材料（如薄荷醇、PEG）须在实验室可去除
> - **时效性**：出土后数小时内完成应急处理
> 
> **技术体系**
> - **微环境控制**：饱水漆木器**原位保湿**（PEM膜、水雾喷淋）；金属锈蚀层**密封避氧**（除氧剂+铝塑袋）
> - **临时固型提取**：脆弱遗存**薄荷醇固型**（可逆）或**石膏绷带套箱**（整体搬运）
> - **现场快检筛查**：便携式XRF识别青铜"粉状锈"（含Cl），便携式拉曼鉴定颜料稳定性，指导提取优先级
> 
> **功能定位**
> 作为**田野考古与实验室修复的临界接口**，现场保护将"抢救性修复"前移至发掘现场，通过维持遗存**物理完整性**与**信息完整性**，确保地层学、类型学及后续科技分析（C14、DNA）的样品有效性。其技术重心是**"延缓"**而非"治愈"，为实验室科学保护争取安全转移时间。
> 

## 与田野考古的关系

| 维度       | 田野考古（Field Archaeology） | 考古现场保护（Archaeological Field Conservation） |
| -------- | ----------------------- | ----------------------------------------- |
| **范畴**   | 母系统（全流程）                | 子系统（技术环节）                                 |
| **工作阶段** | 调查→勘探→发掘→记录→资料整理        | 贯穿发掘**全过程的技术介入**                          |
| **核心目标** | 提取考古学信息（层位、遗物、遗迹）       | 保障遗存完整性（延缓劣化、安全转移）                        |
| **人员主体** | 考古领队、发掘技工、绘图员           | **文物保护师**（Conservator）、科技考古人员             |
