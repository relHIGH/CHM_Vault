# 遗址保护
[[考古学范式#公共考古]]
- 背景
    - 考古遗址具有历史文化价值
    - 为精神文明建设
    - 不可再生
    - 与经济建设，社会发展的关系
- 现状
    - 管理体制研究不够深入
    - 没有形成相对独立的有效运行机制和管理体系
    - 缺乏技术性保护
    - 教育和展示功能缺陷
    - **管理困境**
		- 城乡冲突（遗址区城市建设矛盾）
		- 土地权属（遗址土地与农民/开发商矛盾）
		- 社区参与（原住民安置、利益共享）
		- 展示阐释（土遗址可读性、价值传达）
- 将来
    - 科学，有效，可持续的保护管理
    - 遗址及背景环境的协调问题
    - 积极保护，合理利用是更高层次的保护
    - 人人保护，人人共享文化遗产资源原则
    - 保护理念
        - 专门专业保护到社会参与保护
        - 单一行业到政府各级
        - 被动限制性保护到主动引导性保护
- 新路径
    - 国家大力支持文化遗产保护
    - 新理念新机遇
    - 国际交流

> - 《巴拉宪章》定义的"文化意义"（Cultural Significance）包含哪四个层面？  
> 	- **社会/精神价值**（记忆、情感）★
> 	- **历史价值**（时代见证）
> 	- **科学/考古价值**（信息源）
> 	- **美学价值**（艺术审美）  
> **易错点**：区别于《中国文物古迹保护准则》的"历史、艺术、科学价值"三分法

- **保护手段**
	- 原址回填（Reburial/Covers）
	- 原址罩护/建馆（Shelter/Museum on Site）
	- 迁移保护（Relocation）
	- 遗址博物馆（原位展示、不可移动展品）
	- 生态博物馆[[博物馆案例#贵州梭戛苗族生态博物馆]]
- **科技应用**
	- 微环境控制（温湿度、冻融循环）
	- 加固材料（PS材料、纳米氢氧化钙、有机硅）
	- 监测预警（沉降监测、裂隙监测、游客承载量）
	- 数字化展示/遗址史迹陈列馆[[博物馆案例#数字敦煌]]

# 大遗址保护区 (Strict Protection Zone）
2010 从"保点"到"保区域"
- 大遗址概念**判定标准**（三要素同时具备）：
	- 已列入全国重点文物保护单位
	- 占地≥5平方公里（或文化堆积丰富）
	- **现状以农业农村环境为主**（区别于城市考古遗址）
[[遗址#唐长安城]] [[遗址#英国巨石阵]] [[遗址#安阳殷墟宫殿宗庙区核心区]]



> [!tip] 大遗址
> **【学术定义句】**  
> 大遗址是指规模宏大、价值重大、影响深远的古代文化遗址及遗址群，通常涵盖大型聚落、都城、陵寝、窑址、矿冶遗址等不可移动文物，是中华文明起源与发展的核心物质载体。
> 
> **【核心要素拆解】**  
> - **空间规模**：占地面积通常超过百万平方米，格局清晰，遗存丰富（如汉长安城遗址36平方公里、秦始皇陵56平方公里）；  
> - **遗产价值**：具有重大历史、科学、艺术价值，常为历代帝陵、都城、大型手工业遗址，是阐释中华文明“多元一体”格局的关键实证；  
> - **保护特征**：保护范围与建设控制地带涉及大片土地，与当代城乡发展（城镇化、基础设施建设）矛盾尖锐，保护任务艰巨；  
> - **管理形态**：通常以**考古遗址公园**或**国家文化公园**形式进行整体保护展示，强调**原址保护**与**环境完整性**。
> 
> **【学术定位句】**  
> 该概念标志着中国文化遗产保护从单体文物保护向**片区整体保护**、从被动抢救向**主动规划**的转型，是“中华文明探源工程”与“考古中国”重大项目的核心对象，但也面临**土地权属复杂**、**保护与发展冲突**、**展示阐释难度大**（土遗址可读性低）等现实困境。
> 


# 考古遗址公园 （Archaeological Park）
- 国家考古遗址公园（中国的等级化制度）
**概念**：中国特有的**分级认证制度**（2010年启动），并非简单的"遗址公园"，而是需经国家文物局评定挂牌的**最高等级**考古遗址保护利用模式。

- 功能分区规划（大遗址专项）
	1. **遗址展示区**（原址展示、覆盖保护棚）
	2. **博物馆区**（遗址博物馆/数字展示馆）
	3. **服务配套区**（游客中心、停车，须**外迁**或位于缓冲区边缘）★
	4. **考古预留区**（暂不发掘、原状保护）
**规范依据**：国家文物局《大遗址保护利用导则（试行）》
[[遗址#秦始皇陵]] [[遗址#意大利庞贝]] [[遗址#圆明园遗址公园]]



> [!note] 论述考古遗址公园建设中保护与利用的张力及协调路径
> 【范式】借用**批判性遗产研究（Critical Heritage Studies）**视角，考古遗址公园并非中立的保护技术，而是**遗产政治**的空间实践——涉及**文化资本分配**（谁受益？游客还是原住民？）、**空间权力**（谁的记忆被展示？）与**发展主义**（GDP vs 原真性）的多重博弈。
> 
> 【史脉】中国遗址公园发展三阶段：①**简单绿化期**（1990s，仅圈地植树，无展示）；②**公园化建设期**（2000s，铺广场、建大门，**建设性破坏严重**）；③**考古遗址公园规范化期**（2010《考古遗址公园管理办法》出台，强调**科研优先、考古先行**）。
> 
> 【张力】
> 
> - **原真性 vs 可读性**：土遗址"颜值低"，为吸引游客而**复建仿古建筑**（如大唐芙蓉园），违背《威尼斯宪章》可识别性。
>     
> - **保护需求 vs 土地财政**：遗址公园多位于城市边缘，地方政府借"文化公园"名义**征地卖地**，导致遗址环境碎片化。
>     
> - **专家话语 vs 社区权益**：原住民被迁出（破坏"活态遗产"），或被迫从事低层次旅游服务，**文化受益权不均**。
>     
> 
> 【协调路径】
> 
> - **从"展示废墟"到"阐释价值"**：不以视觉奇观为目标，而通过**考古工作站开放**、**数字虚拟展示**（AR还原）提升可读性，减少实体干预。
>     
> - **社区考古嵌入**：保留/回迁原住民，将其转化为**遗址守护者**（如良渚遗址的"护景员"），共享门票收益（**文化正义**）。
>     
> - **分区管理**：严格划分**考古保护区**（不开放）、**缓冲区**（科研）、**展示区**（游憩），实施**游客承载量控制**（预防性保护）。
>     
> 
> 【批判】考古遗址公园不应是**"主题公园"**（Theme Park）的考古版，而应回归**"Site"**的本义——作为**科学研究的现场**与**文化记忆的圣所**。其终极伦理是**"让遗址活下去"**（Sustainable Conservation），而非**"让遗址活起来"**（商业活化）的过度消费。
>
