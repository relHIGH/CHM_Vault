- 可能题型  
    ①论文化遗产保护与利用的辩证关系  
    ②文旅融合背景下遗产活化路径  
    ③真实性原则在利用中的边界  
    ④地方经济发展与遗产保护的协调机制


- 理论原则  
	[[文物学#文物保护原则]]
    真实性、完整性、最小干预、可持续性、公众共享。  
    强调“保护优先、合理利用”。
    
- 现实冲突  
    过度商业化、空间改造破坏原真性、符号化消费、社区利益失衡、承载量超限。
    
- 治理路径  
    分级保护制度、功能置换而非结构改造、容量控制、收益反哺保护、社区参与机制、数字化替代展示。
    
- 答题结构  
    先界定保护与利用的张力，再分析冲突成因，最后提出制度化解决方案。
