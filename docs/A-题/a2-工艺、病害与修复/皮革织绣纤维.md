# 织绣品类
 #礼制 
- 材质
	棉麻丝毛
- 原料处理（取丝/麻/棉→清洗、分梳） → 纺纱（加捻成纱） → 整经（经线排列） → 上机织造 → 后整理（平整、染整）
- 技法
	- **经锦**  
	    以**经线起花**，纬线为地；结构稳定，纹样规整，多见早期锦。
	    
	- **纬锦**  
	    以**纬线起花**，经线为地；色彩丰富，装饰性强，汉唐常见。
	    
	- **缂丝**  
	    通经断纬，**像“织出来的画”**；工序最复杂，等级最高。
	    [[可移动器物#吴越宋元丝织品]]
	- **织金锦**  
	    在锦中**加入金线**（贴金/包金）；强调礼仪与权力象征。

- 织造
	- 平纹：  
	    经纬一上一下交织，结构最紧密，最耐用
	    
	- 斜纹：  
	    交织点呈斜线，柔软度与强度居中
	    
	- 缎纹：  
	    缎纹组织，**交织点少、表面光滑**；重光泽，不以起花为主。

- 刺绣
	平绣、锁绣、盘金绣、套针绣等



> [!example] 织绣与礼制
>**T｜定位（是什么、在体系中的位置）**  
> 织绣是中国古代礼制体系中重要的物质载体之一，属于通过服饰制度表达等级秩序与社会规范的核心手段。相较于青铜礼器、玉器等在特定仪式中使用的礼制器物，织绣直接作用于日常服饰与礼仪场合，是礼制在社会生活中最直观、最持续的呈现形式。
> 
> **R｜分述（为什么、怎么发挥作用）**  
> 第一，织绣通过**织造结构与工艺复杂度**实现等级区分。不同等级在平纹、斜纹、缎纹以及锦、缂丝、织金锦等工艺的使用上存在明确差异，高复杂度工艺意味着更高的社会投入与制度许可。  
> 第二，织绣通过**纹样与色彩制度化**体现礼制规范。纹样题材、色彩组合及是否使用金线，均受到礼制与法律约束，构成可识别的身份标志，防止僭越。  
> 第三，织绣通过**使用场合的限定**强化礼制秩序。在祭祀、朝会、婚丧等不同礼仪场景中，服用织绣的等级、形式与装饰均有严格区分，使礼制规范贯穿社会运行的多个层面。
> 
> **E｜例证（用具体材料支撑）**  
> 从考古与文献材料看，汉唐以来的锦、织金锦多用于高等级服饰与礼仪场合，而缂丝因工序极繁、成本极高，长期作为高等级陈设或礼仪用物存在。这类织绣并非单纯审美追求，而是制度许可下的结果，反映了礼制对技术与材料的直接调控。
> 
> **E｜小结（回扣题干）**  
> 因此，织绣并非附属于礼制的装饰现象，而是通过工艺、纹样与制度约束，将抽象礼制转化为可穿戴、可识别的社会秩序。

# 皮革



> [!tip] **鞣制技术**  
指将动物皮经过去肉、脱毛后，通过植物性或矿物性鞣剂处理，使皮中胶原纤维发生稳定变化，从而增强柔韧性、耐久性并防止腐朽的工艺，是皮革由生皮转化为可长期使用材料的关键技术。

# 保护修复
病害类型：霉变、脆化、褪色、虫蛀、水解

纺织品修复
	除污-消毒-清洗-平整-加固-防尘-防霉-避光
	丝网加固
	玻璃夹封
	针线

保护原则：恒温恒湿、避光、防虫，最小干预
稳定优先于修复

- **皮革**
    
    - 重点问题：干缩、脆化、粉化
        
    - 核心措施：**缓慢回湿**、柔性支撑
        
    - 禁忌：水洗、油脂随意涂抹
        
    - 逻辑：防止胶原纤维进一步断裂
        
- **织绣 / 纤维**
    
    - 重点问题：断裂、褪色、强度下降
        
    - 核心措施：**加衬、支托、平面固定**
        
    - 修复材料必须低应力、可拆除
        
    - 禁忌：拉伸、重新上色

技术趋向
- **材料升级**：  
    使用**高稳定、低老化的合成纤维与凝胶**作支托与加固，避免传统材料二次伤害。
    
- **无水/低水技术**：  
    针对皮革、丝织品，发展**控释回湿、凝胶清洁**，避免整体水处理。
- **纳米纤维支托（electrospun nanofiber supports）**
    
    - 用于丝织、棉织加固
        
    - 超薄、同尺度、低应力，**几乎不改变手感与外观**
        
- **可控释放凝胶清洁（agar / gellan / nanogel）**
    
    - 精准局部清洁
        
    - 控水、可回收，避免传统“湿处理”风险
        
- **胶原稳定化处理（collagen re-stabilization）**
    
    - 皮革专用
        
    - 针对胶原降解机理，而不是表面“上油”
        
- **微环境封装展示（sealed microclimate display）**
    
    - 修复与展示一体化
        
    - 通过被动缓冲材料长期稳定状态
        
- **数字预修复（digital mock-up before intervention）**
    
    - 修复前在数字模型中测试方案
        
    - **减少一次性、不可逆试错**
