
# 石窟石刻
阴山岩画
（新石器—近代，内蒙）：北方游牧民族的“无字史书”。
_凿刻/磨刻工艺、狩猎游牧场景、原始宗教、北系岩画代表_

沧源岩画
（新石器时代，云南）：我国南方原始艺术的典型代表。 
_赤铁矿涂绘、干栏式建筑、原始生活图谱、南系岩画代表_ 

###### 尕日塘秦刻石 
（秦代，青海兴海县）：2004年发现于青海省兴海县尕日塘，是目前已知秦代遗存分布的最西端，实证了秦帝国对河源地区的有效管控。
*核心遗迹：刻石位于黄河上游阶地的古城址附近。文字风格为典型的小篆，内容涉及秦始皇巡狩、统一文字及度量衡等功绩。
工艺特征：书体严谨，笔画圆转，具有李斯小篆的典型神韵，与泰山、琅琊刻石风格高度一致。
学术价值：打破了“秦始皇从未到达青海”的传统认知；实证了秦帝国在统一后对西部边疆（西羌地区）的军事威慑与行政宣示；是研究秦代疆域、巡狩制度及书法史的极珍贵实物。* 
**新证据**：在刻石附近发现了配套的秦代建筑基址和陶片，证明这里不是孤零零的一块石头，而是一个具有**祭祀或行政功能**的遗址点。  #边疆考古 

喀什三仙洞
（东汉—清，新疆疏勒）：中国地理位置最西部的石窟寺。 
_早期造像特征、单室窟、犍陀罗艺术影响、疏勒早期佛教实证_  #边疆考古 

克孜尔石窟
（东汉—唐，新疆拜城）：中国地理位置最西的大型石窟群。最早凿建,影响了后来云冈和敦煌的早期形制。
*龟兹风、菱格画、最早期石窟、人体比例精确* #边疆考古 

连云港孔望山摩崖石刻
（东汉，江苏连云港）：中国最早的佛教造像遗存。
*佛道杂糅、汉画风格、早于“陆上丝路”的实证* #长江下游 ^35a7f4

云冈石窟
（北魏，大同）：中国早期石窟艺术的巅峰。
*昙曜五窟、胡风汉韵、皇家造像气派*

龙门石窟
（北魏—唐，洛阳）：书法艺术与石刻的完美结合。
*龙门二十品、奉先寺卢舍那大佛、中原风格*

七个星石窟
（晋—元，新疆焉耆）：古代焉耆国规模最大的佛教遗迹。 
_地面寺院与石窟结合、哈拉沙尔风格、佛像陶塑、西域与中原艺术交汇_

（南北朝—元，新疆吐鲁番）：回鹘高昌时期的皇家寺院。

# 墓葬

## 秦始皇祖母陵（夏太后墓）
（秦代，西安长安区）：秦帝国高等级陵寝建筑的典型代表。
*核心发现：在陪葬坑中发现了一种已灭绝的长臂猿新物种（命名为“帝国君子长臂猿”）。
学术价值：通过生物考古展示了秦代关中地区的生态环境多样性，同时反映了秦王室高级随葬宠物的丧葬习俗。

## 海昏侯墓
（西汉，江西南昌）：第一代海昏侯刘贺的墓葬，汉代高等级丧葬制度的缩影。
*核心发现：首次出土秦汉时期的全本《诗经》文本（竹简），以及海量金器、青铜器、马蹄金、刘贺印章。
学术价值：简牍的发现对经学研究具有划时代意义，实证了汉代儒家经典的流传与演变。

狮子山楚王墓
（西汉，徐州）：早期凿山为穴墓典型。
*金缕玉衣、兵马俑坑、崖洞墓* #长江下游

满城汉墓
（西汉，保定）：中山靖王刘胜墓。
*长信宫灯、错金博山炉、崖墓*

###### 北园东晋墓
（东晋时期，山东济南）：北方地区极为罕见的、带有明确南方文化色彩的高等级东晋墓葬。
*形制布局：双室砖筑墓，由墓道、甬道、前室、侧室和后室组成，规模宏大，结构严谨。
核心发现：保存完好的精美壁画。题材包括：出行图、居家生活图、庖厨图，以及具有典型时代特征的“升仙”与“儒家礼仪”融合图案。
出土遗物：青瓷器（具有明显的南方瓷窑风格）、金银饰品、成组的陶俑、成套的饮食器具。
学术价值：实证了东晋时期南北方文化的交流与互动；为研究当时北方士族在动荡局势下的丧葬习俗、生活方式及服饰制度提供了极其珍贵的实物图像资料。* #南京地域

###### 西夏陵
（北宋—南宋/西夏时期，宁夏银川）：西夏历代皇陵所在地，是中国现存规模最大、地面遗迹最完整的帝王陵园之一。
*形制布局：深受中原汉唐陵制影响，又具党项族特色。每座陵寝由阙台、碑亭、月城、献殿、灵台（墓冢）组成。
核心遗迹：九座帝陵与200多座陪葬墓。其灵台为夯土筑成的八角或六角形塔式建筑，外观独特。
典型出土：西夏文碑残片、迦陵频伽（妙音鸟）建筑构件、精美的琉璃瓦、石像生。
学术价值：实证了西夏“仿效中原、保持民族特色”的文化传统；其独特的陵墓空间布局与塔式灵台，是研究西夏宗教、艺术及民族交融史的物证。*   #边疆考古
